# 2025-Vibe-Coding-jian-hao-Wang
### 設計理念：我想創造美式的氛圍感
### 關鍵提示詞(Prompt)：角色設定：「你現在是一位專業的前端工程師與 UI 設計師」 具體目標：「我想做一個美國的旅遊攻略網站，其中包括完整七天六夜的移動、美食、景點攻略」 視覺風格 (Vibe)：「風格要美式、硬漢，使用美國相關色彩與圓角設計」
### 問題解決紀錄：在製作旅遊攻略網站時，最大的困難是如何在主頁同時呈現大量行程資訊，又不讓使用者感到混亂，一開始放太多文字與內容，畫面顯得複雜、不易閱讀。我透過與 AI 討論後，將內容重新整理成「東岸、西岸、跨州、公園」等不同方案，改用卡片式介面呈現，讓使用者可以快速比較與選擇，同時調整視覺風格與互動方式，使整體介面更加清楚、易用且符合主題風格。
### 自我心得：這次專案是我第一次透過 Vibe Coding 的方式完成作品，從整體風格、版面氛圍到功能規劃，都是先確立感覺與使用體驗，再逐步落實成實際介面與功能。這種開發方式讓我更有方向，也更容易調整設計，讓作品同時兼顧美觀與實用，對我而言是一次很有收穫的學習經驗。
